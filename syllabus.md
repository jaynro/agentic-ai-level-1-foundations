# Master Syllabus – 12-Week Agentic AI Developer Program

Below is the week-by-week syllabus. Each week lists the module topic (with the ADK agent or focus in parentheses), along with official learning resources from Google Cloud Skills Boost and Microsoft Learn to solidify that week’s content.

| **Week**    | **Module & ADK Focus**                                         | **Google Cloud Skills Boost – Course/Skill**                                                                                    | **Microsoft Learn – Module/Path**                                                                                |
| ----------- | -------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| **Week 1**  | Foundations of Generative AI *(Intro to ADK)*                  | Introduction to Generative AI – Google Cloud (30 min course). Covers generative AI basics, use cases, and Google’s GenAI tools. | Introduction to Generative AI and Agents *(Beginner, ~1 hr)*. Fundamentals of LLMs, prompts, and agent concepts. |
| **Week 2**  | LLMs and Prompt Engineering *(LLM Agent usage)*                | Introduction to Large Language Models – Google Cloud (1 hr course). Prompt tuning, Gemini tools.                                | Prompt Engineering Fundamentals *(Part 4 of Generative AI for Beginners)*. Best practices with Azure OpenAI.     |
| **Week 3**  | Agent Architecture 101 *(Sequential workflows)*                | Building AI Applications with GenAI – Cloud Skills Boost Path. Multi-step agent apps.                                           | Develop an AI App with Azure Foundry SDK *(Module 3)*. Build GenAI apps with custom logic.                       |
| **Week 4**  | Capstone Project 1 Development *(Fact-Checker)*                | Challenge Lab: Generative AI Practitioner. Apply GenAI basics.                                                                  | *(No new module)*. Project 1: Fact-Checking Auditor – Build & demo a finance domain agent.                       |
| **Week 5**  | Multi-Agent Design Patterns *(SequentialAgent, ParallelAgent)* | Google ADK Multi-Agent Patterns – Google Developers Blog. 8 agent team patterns.                                                | Plan and Prepare AI Solutions on Azure *(Foundry Module 1)*. Architecture planning for AI systems.               |
| **Week 6**  | Tool Integration & Plugins *(ADK Tools & Connectors)*          | Building Tools for AI Agents – Google Cloud Lab. Connect APIs and tools.                                                        | Enable Azure OpenAI with External Data. RAG with your data on Azure.                                             |
| **Week 7**  | Responsible AI & Agent Safety *(Evaluation & Guardrails)*      | Introduction to Responsible AI – Google Cloud (30 min). Bias, fairness, safety.                                                 | Implement a Responsible GenAI Solution *(Foundry Module)*. Mitigate harmful content and bias.                    |
| **Week 8**  | Capstone Project 2 Development *(Support Bot)*                 | Skill Badge Lab: Build a GenAI Support Chatbot. Vertex AI + ADK.                                                                | *(No new module)*. Project 2: Customer Resolution Squad – Retail support agent system.                           |
| **Week 9**  | Autonomous Agents & Self-Healing *(LoopAgent)*                 | ADK Looping Agents Codelab. Build autonomous repair agents.                                                                     | Fine-Tune and Deploy in Foundry *(Optional Module)*. Customize AI behavior.                                      |
| **Week 10** | Testing & Iteration Strategies *(Spec-Driven Dev)*             | Spec-Driven Dev Tutorial (GitHub/Blog). SPECs + test + refine.                                                                  | DevOps for AI: Monitor & Evaluate. Add AI to CI/CD with testing.                                                 |
| **Week 11** | Deployment & Scaling Agents *(Vertex AI, Cloud Run)*           | Skill Badge: Deploy an Agent with ADK. End-to-end multi-agent deployment.                                                       | Evaluate GenAI Performance *(Foundry Module)*. Optimize deployed agents.                                         |
| **Week 12** | Capstone Project 3 Finale *(Self-Healing DevOps)*              | Google Innovators Live: AI Agent Showcase *(optional)*                                                                          | *(No new module)*. Project 3: Self-Healing Engineer demo and review.                                             |

**Note:** Throughout the syllabus, *Google Cloud Skills Boost* activities provide hands-on labs and skill badges (free credits via Innovators program), while *Microsoft Learn* modules offer complementary perspective on generative AI and agent solutions in Azure. Each week builds toward the monthly capstones, ensuring that by Weeks 4, 8, and 12 you are ready to deliver each project.
